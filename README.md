# DFHome — интеграция «Демо»

Официальная демо-интеграция для [DFHome](https://github.com/DFHome/core): виртуальные
устройства, комнаты, виджеты дашборда и план дома. Установите из магазина, чтобы
увидеть приложение без реального оборудования.

## Установка

Через магазин DFHome (курируемый индекс [DFHome/store](https://github.com/DFHome/store))
или вручную — «Добавить репозиторий» с URL этого репозитория.

Рекомендуемая версия: тег **`v1.0.0`**.

## Структура

```
manifest.json   # метаданные для ядра и магазина
__init__.py     # setup() / unload()
data.py         # все демо-данные (устройства, комнаты, виджеты, план)
```

Весь демо-контент живёт в `data.py` — это эталон для правила «всё демо — в demo».

## Зависимости

Интеграция импортирует `app.core.*` из ядра DFHome. Отдельные Python-пакеты не
требуются (`requirements` в манифесте пустой).

## Версионирование

Версии — Git-теги SemVer (`v1.0.0`, `v1.1.0`, …). Ядро ставит закреплённый `ref`
из индекса магазина.
